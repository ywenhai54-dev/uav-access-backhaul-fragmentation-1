# Metrics

This project evaluates UAV-BS deployment using access-backhaul serviceability metrics.

## POI Coverage
Measures whether a point of interest can be covered by at least one selected UAV-BS.

## Binary Joint Serviceability
Measures whether a POI is jointly serviceable through both access connectivity and backhaul reachability.

## Capacity-Constrained JSWPR
Extends binary serviceability by considering backhaul capacity limits and service-chain bottlenecks.

## Fragmentation Indicators
The project also supports graph-based indicators such as grounded spectral connectivity, graph resistance, diameter, terrain fragmentation index, and line-of-sight ratio.
